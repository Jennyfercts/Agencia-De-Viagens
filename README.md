# Agência De Viagens

Protótipo simples desenvolvido em Java para automatizar o cadastro inicial de reservas de passagens internacionais de uma agência de viagens.

## Funcionalidades

* Cadastro do nome do passageiro
* Cadastro do número do passaporte
* Seleção do destino da viagem
* Escolha da classe da passagem
* Cálculo automático do valor da viagem
* Exibição de um resumo da reserva

## Tecnologias

* Java
* JOptionPane (Swing)

## Objetivo

Praticar conceitos de programação em Java, como entrada e saída de dados, estruturas condicionais e cálculos, por meio do desenvolvimento de um sistema simples de reservas.
